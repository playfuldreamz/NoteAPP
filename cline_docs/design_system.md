# Design System

## Colors
- Primary: #3b82f6 (blue-500)
- Secondary: #6b7280 (gray-500)
- Background: 
  - Light: #ffffff
  - Dark: #1f2937 (gray-800)
- Text:
  - Light: #111827 (gray-900)
  - Dark: #f3f4f6 (gray-100)

## Typography
- Font Family: System UI
- Heading Sizes:
  - h1: 2rem
  - h2: 1.5rem
  - h3: 1.25rem
- Body Text: 1rem

## Spacing
- Base Unit: 0.25rem
- Common Spacings:
  - Small: 0.5rem
  - Medium: 1rem
  - Large: 1.5rem
  - X-Large: 2rem

## Components

### Cards
- Background: white/dark:gray-800
- Border: 1px solid gray-200/dark:gray-700
- Border Radius: 0.75rem
- Padding: 1.5rem
- Shadow: sm

### Buttons
- Primary:
  - Background: blue-500
  - Text: white
  - Hover: blue-600
- Secondary:
  - Background: gray-50/dark:gray-700
  - Text: gray-700/dark:gray-300
  - Hover: gray-100/dark:gray-600

### Lists
- Item Padding: 0.5rem
- Hover Background: gray-50/dark:gray-700
- Active Background: blue-50/dark:blue-900
